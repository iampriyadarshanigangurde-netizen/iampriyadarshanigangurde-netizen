### things I've built

#### 🔐 IoMT Intrusion Detection *(Master's Thesis)*
> 🏆 **2nd Prize — Fordham CIS Research Exhibition 2025**

Protocol-aware ML system on the CICIoMT2024 dataset — **7.8M records, 48 features**.  
Benchmarked 8 classifiers across 4 class-balancing strategies.  
Top model: **99.81% accuracy · Macro F1 = 0.97**  
Balanced Random Forest hit **98% recall** on rare MQTT and Spoofing attacks.

---

#### 🛒 Customer Segmentation & Product Recommendation
*Online retail · 392K transactions · Scikit-learn · TensorFlow · mlxtend*

Built a full recommendation pipeline from scratch:
- **RFM analysis** → behavior scoring
- **K-Means** (4 segments, Silhouette = 0.41) → customer clusters  
- **Market Basket / Apriori** (1M+ rules, 94% statistically significant) → item associations  
- **Neural Network** collaborative filter → personalized product suggestions

---

#### 🎴 AI Flashcard Generator
*LangChain · Vertex AI · FastAPI · YouTube Transcript API*

An AI teaching assistant that converts YouTube videos into structured flashcards — Q&A pairs, context-aware, ready to study.  
Architecture: URL → transcript → tiktoken preprocessing → LangChain + Gemini → JSON flashcard output.  
Published as an [academic paper](mailto:prg@fordham.edu) at Fordham Graduate School.

---

#### 🚗 CO₂ Emissions Prediction
*7,385 vehicle records · Linear Regression · Random Forest · Plotly*

Predicted tailpipe CO₂ from engine features (size, cylinders, fuel consumption).  
Cleaned 1,103 duplicates, engineered transmission and fuel type categories, removed outliers via Z-score.

| Model | R² | RMSE (test) |
|---|---|---|
| Linear Regression | 0.906 | ~15.4 |
| **Random Forest** | **0.973** | **~10.4** |

Full EDA-to-stakeholder pipeline with Plotly visualizations.
